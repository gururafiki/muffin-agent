## Package extras

- [OpenSandbox](https://open-sandbox.ai/) - used for secure execution of code generated by agent
- [OpenBB](https://openbb.co) - used as a proxy for various financial data sources
- [Agent Chat UI](https://github.com/langchain-ai/agent-chat-ui) - UI for interacting with Agent


### Additional references

#### Langchain skills installation
```bash
npx skills add langchain-ai/langchain-skills --skill '*' --yes
```

#### Langsmith CLI installation
```bash
curl -sSL https://raw.githubusercontent.com/langchain-ai/langsmith-cli/main/scripts/install.sh | sh
```

#### Langsmith skills installation
```bash
npx skills add langchain-ai/langsmith-skills --skill '*' --yes
```

#### Langfuse skills installation
```bash
npx skills add langfuse/skills --skill "langfuse"
```
