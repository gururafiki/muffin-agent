## Package extras

- [OpenSandbox](https://open-sandbox.ai/) - used for secure execution of code generated by agent
- [OpenBB](https://openbb.co) - used as a proxy for various financial data sources
- [Firecrawl](https://github.com/firecrawl/firecrawl) - self-hosted web scraping and crawling service. Requires `firecrawl-redis`, `firecrawl-rabbitmq`, `firecrawl-playwright`, `firecrawl-postgres` (nuq-postgres image with pg_cron), and `firecrawl-api` (harness.js --start-docker). Config in `extras/firecrawl/.env`.
- [SearxNG](https://github.com/searxng/searxng) - self-hosted meta-search engine aggregating Google, Bing, DuckDuckGo, Brave, and others. Config in `extras/searxng/`. Used directly by agents and as a backend for Firecrawl's search endpoint.
- [Agent Chat UI](https://github.com/langchain-ai/agent-chat-ui) - UI for interacting with Agent
