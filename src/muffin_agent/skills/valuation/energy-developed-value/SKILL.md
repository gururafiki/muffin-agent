# Energy — Developed Markets Value Strategy

**Primary Method:** Free Cash Flow Yield + Production Multiples + Reserve Assessment

## Key Metrics

| Metric | Target | Assessment |
|--------|--------|------------|
| **FCF Yield** | 6-10%+ | Oil majors (Saudi Aramco 5%, Petrobras 27.8%) |
| **Production Volume** | Stable/growing | Reserve life >10 years |
| **Cost per Barrel (TAC)** | Competitive vs WTI price | Total Average Cost model |
| **Reserve Replacement Ratio** | >1.0x | Proving new reserves to offset production decline |
| **Dividend Yield** | 4-8% | High payout ratio acceptable (cyclical business) |
| **Free Cash Flow Conversion** | 60-80% | Capital discipline; reinvestment needs |
| **Debt/EBITDA** | <2.0x preferred | Lower leverage for commodity cyclicals |
