# Telecommunications — Developed Markets Value Strategy

**Primary Method:** EV/EBITDA + ARPU Analysis + Dividend Yield + Customer Metrics

## Key Metrics

| Metric | Target | Assessment |
|--------|--------|------------|
| **EV/EBITDA** | 6-8x | Mature telecommunications multiple |
| **ARPU** | $40-70/month (varies) | Average Revenue Per User; market mature |
| **ARPU Trend** | Flat to +2% YoY | Declining volume offset by pricing |
| **Customer Churn** | 20-30% annual | Industry typical; varies by market |
| **Dividend Yield** | 3-5% | Stable distributions; part of total return |
| **ROIC** | >8% minimum | Capital intensity high; moderate returns |
| **Debt/EBITDA** | 2.5-3.5x | Leverage acceptable; capex-heavy |
| **Capex Intensity** | 15-20% of revenue | 5G upgrade cycle ongoing |
