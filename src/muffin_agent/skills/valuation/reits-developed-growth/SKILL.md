# REITs — Developed Markets Growth Strategy

**Method:** FFO/AFFO Growth + NOI Growth + Cap Rate Compression

## Key Metrics

| Metric | Target | Driver |
|--------|--------|--------|
| **FFO Growth** | 5-10% annually | Organic NOI growth + acquisitions |
| **NOI Growth** | Positive 3-5%+ | Rent growth + occupancy gains + operational efficiency |
| **Occupancy Rate** | >90% | Leasing momentum; portfolio quality |
| **Tenant Credit Quality** | Grade A dominant | Payment reliability |
| **Cap Rate Compression** | 25-50 bps annually | Property value appreciation |
| **Dividend Growth** | 2-4% annually | Supported by FFO growth |
| **AFFO Payout Ratio** | 80-90% | Sustainable distribution + retention |
