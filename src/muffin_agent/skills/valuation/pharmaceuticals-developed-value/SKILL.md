# Pharmaceuticals — Developed Markets Value Strategy

**Primary Method:** P/E Ratio + Risk-Adjusted NPV (rNPV) of Pipeline

## Key Metrics

| Metric | Target | Assessment |
|--------|--------|------------|
| **P/E Ratio** | 10-15x | Currently at multi-decade lows (Sanofi 12x, Feb 2026) |
| **Operating Margin** | 25-35% | Large pharma typical; smaller biotech 0% acceptable |
| **R&D Efficiency** | 15-25% of revenue | Benchmark; above 25% may indicate pipeline drought |
| **Dividend Yield** | 2-4% | Mature pharmas pay dividends from stable core |
| **Patent Cliff Timeline** | Next 5 years | Critical: map revenue loss by drug |
| **Earnings Quality** | Core vs reported | Adjust for acquisitions, one-time writedowns |

## Critical Assessment

- If pipeline insufficient to replace patent losses, earnings face cliff; P/E should reflect risk
- rNPV of pipeline: risk-weight each drug by its clinical phase probability of success
- Core earnings (excluding one-time items) more relevant than reported EPS
