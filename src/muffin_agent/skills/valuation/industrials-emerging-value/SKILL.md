# Industrials — Emerging Markets Value Strategy

**Method:** EV/EBITDA with EM Discount + Cost Advantage Analysis

## Key Metrics & Adjustments

| Metric | DM Target | EM Target | Note |
|--------|-----------|-----------|------|
| **EV/EBITDA** | 6-10x | 5-8x | 30% discount typical |
| **FCF Yield** | >6% | >7% | Higher required return |
| **ROIC** | >10% | >12% | Higher hurdle rate |
| **Debt/EBITDA** | <3.0x | <2.5x | Less leverage acceptable |
| **Currency Risk** | - | -10-15% adjustment | FX volatility |
