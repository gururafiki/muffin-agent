# Industrials — Developed Markets Value Strategy

**Primary Method:** EV/EBITDA + FCF Yield + ROIC Analysis

## Key Metrics

| Metric | Target | Assessment |
|--------|--------|------------|
| **EV/EBITDA** | 6-10x | Below industry median (~5.4x) = value |
| **FCF Yield** | >6% | Attractive entry point |
| **ROIC** | >10% minimum | Capital allocation quality |
| **Debt/EBITDA** | <3.0x | Balance sheet flexibility |
| **Capex Intensity** | 3-6% of revenue | Equipment/facility replacement needs |
| **Order Backlog** | 6+ months visible | Forward revenue visibility |
| **Gross Margin** | 25-35% (varies by subsector) | Pricing power & cost structure |
