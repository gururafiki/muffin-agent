# Energy — Developed Markets Growth Strategy

**Method:** Production Ramp + FCF Growth + Capital Discipline

## Key Metrics

| Metric | Target | Driver |
|--------|--------|--------|
| **Production Growth** | 5-10% annually | New field development |
| **FCF Growth** | 10%+ annually | Volume + commodity cycle |
| **Cost Reduction** | -$2-5 per bbl annually | Operational efficiency |
| **Reserve Replacement** | 1.5x+ | Building inventory for 10+ year life |
| **Capital Expenditure** | 25-35% of FCF | Reinvestment for growth |
| **Dividend Growth** | 3-5% annually | Supported by FCF |

## Project Economics

- Large projects (>$5B capex) require $60+ Brent for IRR >10%
- Assess: project start date, peak production timeline, cost overrun risks
- Breakeven analysis: what oil price needed for project IRR >12%?
