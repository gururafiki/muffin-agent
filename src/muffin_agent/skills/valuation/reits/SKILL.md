# REIT Valuation — Common Principles

## Why REIT-Specific Metrics

Standard P/E is distorted by depreciation charges. Use instead:
- **FFO** = Net Income + Depreciation - Gain on Sales (recurring operating earnings)
- **AFFO** = FFO - Maintenance CapEx (sustainable distribution capacity)
- **NAV** = Fair Value of Properties - Liabilities (asset-based valuation)

## Cap Rate Analysis

| Property Type | Cap Rate Range | Risk/Return |
|---------------|---------------|-------------|
| **Prime office (CBD)** | 3.5-4.5% | Lowest risk/return |
| **Suburban office** | 5-6% | Moderate |
| **Retail (prime)** | 4-5% | Location dependent |
| **Industrial/logistics** | 3.5-4.5% | High demand |
| **Residential** | 3-4.5% | Stable cash flow |

**Lower cap rate = higher property valuation.** Cap rate compression raises property values.

## NAV Calculation Example

- Property 1: NOI $5M / 5% cap rate = $100M value
- Property 2: NOI $3M / 6% cap rate = $50M value
- Total properties: $150M
- Less: Total liabilities ($50M)
- **NAV = $100M**
