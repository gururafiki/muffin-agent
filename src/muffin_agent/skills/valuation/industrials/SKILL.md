# Industrials/Manufacturing Valuation — Common Principles

## Primary Methodology

EV/EBITDA is the standard industrial valuation multiple because:
- Normalizes for capital structure differences
- Handles varying depreciation policies
- Comparable across subsectors

## Subsector Multiples (2024-2025)

| Subsector | EV/EBITDA Range |
|-----------|----------------|
| **Aerospace manufacturers** | 7.4x-10.9x |
| **Automotive** | 7.0x-10.2x |
| **Food & Beverage** | 8.1x-9.4x (stable) |
| **Industrial IoT** | 7.4x-11.0x (tech premium) |

## FCF Yield Calculation

- FCF Yield = Operating FCF / Enterprise Value
- >6% typically attractive; >8% deep value
- Check FCF quality (not boosted by working capital releases)

## Key Operational Indicators

- Order backlog visibility (6+ months)
- Capex intensity (3-6% of revenue for maintenance)
- Working capital efficiency trends
