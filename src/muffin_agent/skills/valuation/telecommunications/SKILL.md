# Telecommunications Valuation — Common Principles

## ARPU Calculation & Interpretation

- ARPU = Monthly recurring revenue / Average number of users
- Declining ARPU signals: competition, package erosion, churn risk
- Stable ARPU: market equilibrium; pricing power
- Growing ARPU: price increases accepted, 5G adoption driving data ARPU

## Churn Analysis

- Monthly churn 1.5-2.5% typical (annualizes to 18-30%)
- Higher churn: competitive market, service quality issues
- Churn trends more important than absolute level

## Capital Intensity

- Telecom is capex-heavy (15-22% of revenue)
- 5G upgrade cycle ongoing — investment phase
- Returns improve as subscriber base grows on existing infrastructure
