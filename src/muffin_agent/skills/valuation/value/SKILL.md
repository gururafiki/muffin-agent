# Value Stock Valuation Principles

## Core Value Metrics (All Sectors)

- **Operating margin quality critical** — 8-12% minimum depending on sector
- **Free cash flow yield > 6%** — typical entry point for value stocks
- **ROIC > 10%** — benchmark for capital allocation quality
- Quality of earnings: distinguish core vs reported earnings
- Dividend sustainability: payout ratio vs FCF coverage
- Balance sheet strength: debt levels relative to sector norms
