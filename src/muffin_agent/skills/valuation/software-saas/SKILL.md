# Software/SaaS Valuation — Common Principles

## Core SaaS Metrics

| Metric | Significance |
|--------|-------------|
| **PEG Ratio** | Price-to-earnings-growth; normalizes P/E for growth rate |
| **Rule of 40** | Growth Rate + FCF Margin > 40 = balanced health |
| **NRR (Net Revenue Retention)** | Existing customer expansion; >100% = expansion revenue |
| **GRR (Gross Retention Rate)** | Logo retention; 1.5-2.0x ARR multiple premium for top-quartile |

## NRR Deep Dive

- **NRR = (Beginning ARR + Expansion - Churn) / Beginning ARR**
- >100% = expanding from existing customers (very valuable)
- 120%+ NRR commands premium multiples; 10-pt NRR increase = 20-30% valuation uplift
- GRR increasingly important; top-quartile GRR commands 1.5-2.0x ARR multiple premium

## PEG Calculation

- PEG = P/E / Expected Growth Rate
- Company P/E = 25x, expected growth = 25% → PEG = 1.0 (fairly valued)
- If same company trades 22x with 25% growth → PEG = 0.88 (undervalued)
