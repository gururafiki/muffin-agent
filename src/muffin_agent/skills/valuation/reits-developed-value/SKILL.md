# REITs — Developed Markets Value Strategy

**Primary Method:** Price-to-NAV (P/NAV) + FFO & AFFO Multiples + Cap Rate Analysis

## Key Metrics

| Metric | Formula/Target | Significance |
|--------|-----------------|--------------|
| **Net Asset Value (NAV)** | Fair Value of Properties - Liabilities | Base case valuation |
| **Cap Rate** | Property NOI / Market Value | Prime urban: 3-5%; suburban: 6-9% |
| **P/FFO Multiple** | Price / FFO per Share | Primary REIT valuation (12-18x) |
| **P/AFFO Multiple** | Price / AFFO per Share | Closer to distributable cash (14-20x) |
| **Dividend Yield** | Annual Distribution / Price | 3-5% typical |
| **Premium/Discount to NAV** | (Price - NAV) / NAV | Premium = market confidence |
| **Debt/EBITDA** | Total Debt / EBITDA | <5.0x healthy for REITs |
