# Banking Valuation — Common Principles

## Why Dual Metrics (P/B + P/E)

- **P/B alone** can mislead during rapid earnings changes
- **P/E alone** ignores balance sheet quality (critical for banks)
- **P/B + P/E cross-check** provides complete picture

## Core Banking Quality Indicators

| Metric | Significance |
|--------|-------------|
| **Return on Equity (ROE)** | Primary profitability measure; drives P/B multiple |
| **Net Interest Margin (NIM)** | Core income driver; track trends vs rate environment |
| **Non-Performing Loan Ratio (NPL)** | Credit quality signal |
| **Cost-to-Income Ratio** | Operating efficiency |
| **Tier 1 Capital Ratio** | Regulatory buffer and stress resilience |
| **Return on Assets (ROA)** | Operating leverage indicator |
