# Software/SaaS — Developed Markets Value Strategy

**Primary Method:** PEG Ratio + EV/Revenue + Net Revenue Retention (NRR)

## Key Metrics

| Metric | Target | Significance |
|--------|--------|--------------|
| **PEG Ratio** | <1.0 | Stock fairly valued if PEG = 1; <1.0 undervalued |
| **EV/Revenue** | 5-8x | Lower end of SaaS range; growth at 8-15x |
| **Revenue Growth Rate** | 20-30% YoY | Minimum for SaaS to justify premium valuation |
| **Net Revenue Retention (NRR)** | >100% | Existing customers expand usage; expansion revenue key |
| **Gross Margin** | >60% | SaaS standard; software leverage |
| **Rule of 40** | Growth + FCF Margin >40 | Balanced value: 20% growth + 20% FCF margin = healthy |
| **Free Cash Flow Margin** | >15% | Path to profitability |
