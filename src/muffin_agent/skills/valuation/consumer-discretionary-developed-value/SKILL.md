# Consumer Discretionary — Developed Markets Value Strategy

**Primary Method:** P/S Ratio + EV/EBITDA + ROIC Analysis

## Key Metrics

| Metric | Target | Significance |
|--------|--------|--------------|
| **P/S Ratio** | 1.0-2.0x | Below peers = potential value |
| **EV/EBITDA** | 8-12x | Industry baseline |
| **ROIC** | >10% minimum | Capital efficiency; varies by subsector |
| **Gross Margin** | 30-50% (category dependent) | Pricing power & cost structure |
| **Operating Leverage** | Visible | Fixed cost base |
| **Debt/EBITDA** | <3.0x | Balance sheet stress acceptable if normalized |
| **Cash Conversion** | >50% of EBITDA | Working capital management |
