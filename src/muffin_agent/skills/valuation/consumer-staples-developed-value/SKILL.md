# Consumer Staples — Developed Markets Value Strategy

**Primary Method:** P/E + Brand Moat Analysis + Pricing Power Assessment

## Key Metrics

| Metric | Target | Assessment |
|--------|--------|------------|
| **P/E Ratio** | 15-22x | Lower than growth; stability premium |
| **Dividend Yield** | 2-4% | Consistent payout history |
| **Revenue Growth** | 2-5% YoY | Mature market saturation |
| **Gross Margin** | 35-50% | Product mix dependent |
| **Operating Margin** | 12-20% | Efficiency & scale benefits |
| **Free Cash Flow Conversion** | >80% | Capital efficiency |
| **ROIC** | >12% | Returns above cost of capital |
| **Pricing Power Strength** | Demonstrable | Historical price increases accepted |
